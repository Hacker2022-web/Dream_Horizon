# Luxe Interiors - High-End Design Portfolio

This is a responsive, single-page React application built with Vite and Tailwind CSS.

## Project Structure

- **`src/App.tsx`**: Contains all the components and logic for the website (Navbar, Hero, About, Services, Portfolio, Process, Testimonials, Contact).
- **`src/index.css`**: Global styles, Tailwind imports, and Font configurations.
- **`metadata.json`**: Application metadata.

## Features

- **Sticky Navigation**: Smoothly transitions on scroll.
- **Hero Section**: Immersive background with entrance animations.
- **Portfolio**: Filterable gallery with smooth transitions.
- **Services**: Clean 3-column grid layout.
- **Process Timeline**: Vertical step-by-step guide.
- **Testimonials**: Interactive slider.
- **Contact Form**: Professional layout with business details.

## Customization

You can easily customize the content by editing the arrays in `src/App.tsx` (e.g., `projects`, `services`, `testimonials`).
Colors and fonts are defined in `src/index.css` using Tailwind's `@theme` directive.
